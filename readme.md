# Desafío Técnico de Programación Web

## Desarrollo de Simuladores Interactivos con Lógica Matemática

Este proyecto fue desarrollado como parte del desafío técnico de programación web enfocado en lógica matemática, manipulación del DOM y maquetación profesional utilizando tecnologías web.

# Objetivo del Proyecto

El objetivo principal del proyecto es desarrollar simuladores interactivos capaces de resolver problemas matemáticos y físicos mediante JavaScript, implementando formularios dinámicos, validaciones y una interfaz moderna y responsiva.

---

# Ejercicios Desarrollados

## 1. Simulador de Transferencia de Calor

Este simulador implementa la Ley de Enfriamiento de Newton para calcular la temperatura final de un objeto después de un tiempo determinado.

### Fórmula utilizada

T = Ts + (T0 - Ts) * e^(-k * t)

### Características

- Captura dinámica de datos con JavaScript
- Uso de Math.exp()
- Resultado redondeado con Math.round()
- Validaciones de datos
- Diseño moderno y responsivo
- Interfaz adaptada para PC, tablet y móviles

---

## 2. Calculador de Combinaciones Complejas

Este simulador calcula el total de combinaciones posibles para un sorteo utilizando combinatoria matemática.

### Fórmula utilizada

C(n,r) = n! / (r! * (n-r)!)

### Características

- Implementación de factorial recursivo
- Validaciones para evitar valores inválidos
- Cálculo dinámico del resultado
- Resultado formateado automáticamente
- Diseño moderno y responsivo
